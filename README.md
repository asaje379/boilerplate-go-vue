# Boilerplate Go + Vue

A modern fullstack boilerplate: Go backend + Vue frontend.

## 🚀 Quick Start

```bash
npm create asaje-go-vue@latest my-project
cd my-project
asaje start
```

## 📦 Structure

- **`api/`** — Go HTTP API (clean architecture)
- **`admin/`** — Vue 3 + Vite admin frontend
- **`realtime-gateway/`** — Go realtime transport (SSE + WebSocket via RabbitMQ)
- **`pwa/`** — Progressive Web App
- **`landing/`** — Static landing page
- **`cli/`** — Scaffolding CLI
